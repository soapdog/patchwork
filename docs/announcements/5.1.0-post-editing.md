# v5.1.0: Post Editing

The main visible feature for this release is _post editing_. You can now edit the content of your own posts. Editing the content does not overwrite the old content, it just favours showing recent edits over old content. You can still use the little arrows on top an edited post to view all its editing history.

### Changes to EBT

An overridden version of `epidemic-broadcast-trees` is being used now. It adds the format field to the stream and passes "classic" as the value. This field is expected by Tilde Friends and Manyverse.

I'm unsure if not having this field was impacting EBT usage before, EBT is hard to debug. Let me know how this works for yous.
